# 接口契约

## 1. 基本约定

- API 前缀：`/api`
- 接口文档：`http://127.0.0.1:8000/docs`
- 响应格式统一为 `{success, message, data}`

成功响应示例：

```json
{
  "success": true,
  "message": "ok",
  "data": {}
}
```

失败响应示例：

```json
{
  "success": false,
  "message": "错误信息",
  "data": null
}
```

## 3. 登录接口

接口：

```text
POST /api/users/login
```

请求体示例：

```json
{
  "username": "admin",
  "password": "admin123"
}
```

说明：登录成功返回用户基础信息；响应中不返回密码哈希。

## 4. 补货建议接口

生成补货建议：

```text
POST /api/recommendations/generate
```

开启 AI 增强：

```text
POST /api/recommendations/generate?enhance_with_llm=true
```

采用建议：

```text
POST /api/recommendations/{recommendation_id}/accept
```

拒绝建议：

```text
POST /api/recommendations/{recommendation_id}/reject
```
